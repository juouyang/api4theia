# flask-restful-api for theia container

## Prepare

```
git clone https://github.com/juouyang-aicots/api4theia.git
cd api4theia/
python3 -m venv venv/
source venv/bin/activate
venv/bin/pip install -U pip
venv/bin/pip install -r requirements.txt
```

## Run

```
source venv/bin/activate
./app.py
```

## Test

```
source venv/bin/activate
python -m pytest
```
